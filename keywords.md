# Taxonomía de Keywords

Archivo canónico de palabras clave para el Academic Brain. **Toda keyword debe estar aquí antes de usarse en una nota.**

Este archivo se almacena en GitHub para revisión: los cambios son visibles como commits y permiten auditar qué conceptos se agregan y detectar quasi-duplicados antes de que proliferen.

## Cómo agregar una keyword

1. Buscar en este archivo (Ctrl+F) si ya existe una palabra similar
2. Si existe un quasi-duplicado → usar esa forma canónica, no crear una nueva
3. Si es genuinamente nueva → agregarla en la sección alfabética correcta
4. Formato: `**keyword-canónica** — Definición corta. _Aliases: variante1, variante2_`
